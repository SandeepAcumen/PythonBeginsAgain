Q5. What is Integration Runtime?

Integration Runtime is the compute infrastructure used by ADF to perform data movement and transformation.

| Type           | Purpose                  |
| -------------- | ------------------------ |
| Azure IR       | Cloud data movement      |
| Self-hosted IR | On-premises connectivity |
| SSIS IR        | Run SSIS packages        |

Q6. How did you implement incremental load?

    1.We used a watermark column like LastModifiedDate in the source table.
    2.During each pipeline run, we retrieved the last processed timestamp from a metadata table, 
    and only records with a greater timestamp were extracted.

    SELECT *
    FROM Orders
    WHERE LastModifiedDate > @LastRunTime

Q7. What if the source table doesn’t have a timestamp column?

    If no timestamp column exists, I use hash comparison or enable CDC to detect changes.
   Possible solutions:
    1.Change Data Capture (CDC)
    2.Hash comparison
    3.Full load with merge logic
    4.Database triggers

Q8. Difference between Azure Blob Storage and ADLS Gen2?

    ADLS Gen2 provides hierarchical namespace and optimized performance for big data analytics workloads 
    compared to Blob storage.

    | Blob Storage        | ADLS Gen2               |
| ------------------- | ----------------------- |
| Object storage      | Data lake storage       |
| No folder hierarchy | Hierarchical namespace  |
| Limited analytics   | Optimized for analytics |
